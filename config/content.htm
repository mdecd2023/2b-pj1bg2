Note_BubbleRob

1.車輪、感知器...等名稱要與程式相符，否則會出現問題。

2.可調整sensingNose的掃瞄角度、掃描距離，使掃描廣度加大。但要注意不可掃到地面。

3.當迷宮中出現不規則彎道時，若按照原本的程式，會發生繞圈、來回走的狀況。可透過調整倒車時間及前進、後退的速度差來解決此問題，但不同的迷宮會需要不同的最佳化設定。

目前正在嘗試加入第二個sensingNose及第二組程式，使其能夠更快走出迷宮。

雙人連線

BubbleRob雙人對戰.7z

操作原理:

兩人連線到同一主機，同一阜號，分別控制同一場景中的兩隻機器人。

在瀏覽器輸入http://192.168.1.xx:阜號，可看到進行中的場景

(可在CMD輸入以下命令，查看自身IP位址)

ipconfig

圖為雙人對戰場景

以下為玩家1的程式碼

# pip install pyzmq cbor
from zmqRemoteApi import RemoteAPIClient
import keyboard

client = RemoteAPIClient('192.168.1.12', 23000)

print('Program started')
sim = client.getObject('sim')
sim.startSimulation()
print('Simulation started')

def setBubbleRobVelocity(leftWheelVelocity, rightWheelVelocity):
    leftMotor = sim.getObject('/leftMotor1')
    rightMotor = sim.getObject('/rightMotor1')
    sim.setJointTargetVelocity(leftMotor, leftWheelVelocity)
    sim.setJointTargetVelocity(rightMotor, rightWheelVelocity)

'''
# Example usage 1:
setBubbleRobVelocity(1.0, 1.0)
time.sleep(2)
setBubbleRobVelocity(0.0, 0.0)
'''
# use keyborad to move BubbleRob

while True:
    if keyboard.is_pressed('up'):
        setBubbleRobVelocity(1.0, 1.0)
    elif keyboard.is_pressed('down'):
        setBubbleRobVelocity(-1.0, -1.0)
    elif keyboard.is_pressed('left'):
        setBubbleRobVelocity(-1.0, 1.0)
    elif keyboard.is_pressed('right'):
        setBubbleRobVelocity(1.0, -1.0)
    elif keyboard.is_pressed('q'):
        # stop simulation
        sim.stopSimulation()
    else:
        setBubbleRobVelocity(0.0, 0.0)
